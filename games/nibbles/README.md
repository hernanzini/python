# Implementation of [Nibbles](https://en.wikipedia.org/wiki/Nibbles_(video_game))

## Goals: 
* Minimum external libraries
* Minimum source code files and lines
* Easy to read and use

### On Windows:
```cmd
    pip install pygame
```
```cmd
    python nibbles.py  
```
### On Mac:
```bash
    pip3 install pygame
```
```bash
    python3 nibbles.py  
```
